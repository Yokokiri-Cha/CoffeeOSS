# CoffeeOSS4
Android版CoffeeブラウザーのUIを継承した軽量でオープンソースなWindows版のCoffeeブラウザー!4度目の大幅改善!  
## Features  
### ユニークな機能を多数搭載!  
・YouTubeLiveコメント分割表示機能  
・Googleアカウントダイレクト管理ボタン-すぐにアカウント設定が出来ます!  
・STV Emulation-疑似スマートテレビ環境での動画視聴機能  
・Googleサイト翻訳ダイレクトボタン-すぐにサイトを翻訳可能!  
・参考文献リスト作成支援機能-論文作成のお供に!  
・QRコード生成機能-簡単にスマートフォンで閲覧を続行できます!  
・ユーザーエージェント設定ボタン-プリセット値のほか、任意の値もセットできます!  
・DevTools,WebViewタスクマネージャーも搭載!  
・操作音声読み上げ機能(アクセシビリティー)-Android版同様、Coffeeロゴを押すと利用可能です!  
・ミニポップアップウィンドウ  
### 操作感はAndroid版とほぼ同じです!

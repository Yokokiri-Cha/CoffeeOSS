# CoffeeOSS4-オープンソースな多機能ブラウザー!  
## 趣味も調べ物もこれ一つ! 　(..とはいかないかもですが、様々な機能があなたをサポートします!)  
Android版CoffeeブラウザーのUIを継承した軽量でオープンソースなWindows版のCoffeeブラウザー!4度目の大幅改善!  
<img src="https://github.com/Yokokiri-Cha/Resources1/blob/main/img1COSS4.png?raw=true" width="50%">  

## Features  

### ユニークな機能を多数搭載!  
・YouTubeLiveコメント分割,ホップアップ表示機能-配信視聴画面を自由にカスタマイズ!  
・全画面表示(操作ボタンなし)-画面いっぱいにサイトを表示します!  
・Googleアカウントダイレクト管理ボタン-すぐにアカウント設定が出来ます!  
・STV Emulation-疑似スマートテレビ環境での動画視聴機能  
・Googleサイト翻訳ダイレクトボタン-すぐにサイトを翻訳可能!  
・参考文献リスト作成支援機能-論文作成のお供に!  
・QRコード生成機能-簡単にスマートフォンで閲覧を続行できます!  
・ユーザーエージェント設定ボタン-プリセット値のほか、任意の値もセットできます!  
・DevTools,WebViewタスクマネージャーも搭載!  
・操作音声読み上げ機能(アクセシビリティー)-Android版同様、Coffeeロゴを押すと利用可能です!  
・ミニポップアップウィンドウ  
・MiniSearch画面
など  
### Powered by Edge WebView2!-Chromiumの機能を利用できます!

## 履歴機能について  
現時点で、履歴機能やお気に入り登録機能は実装されていません。  
検索履歴を記録したい方はGoogleアカウントでCoffeeブラウザーにログインしてください。  
(ただし、サイト内アクティビティーは記録されません。)  
今後も実装手段を模索していく所存です。方法が見つかり次第、すぐに実装します。  
どうしても履歴記録やお気に入り登録機能をWindowsで使用したい場合はWSAを使用してAndroid版Coffeeをサイドロードできます。  
(WSA:Windows Subsystem for Android)(サイドロード:ここでは手動でアプリをダウンロードしてインストールすることを指します。)
